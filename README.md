# Web Developer Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and web development projects. It is built using plain **HTML**, **CSS**, and **JavaScript** to demonstrate my abilities in frontend development.

---

## 🚀 Features

- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.  
- **Interactive Elements**: Smooth scrolling, dynamic navigation, and animations.  
- **Projects Showcase**: A collection of my completed and ongoing projects with links to live demos and source code.  
- **Contact Form**: A simple form to allow visitors to get in touch with me.  

---

## 🛠 Technologies Used

- **HTML5**: Markup structure.  
- **CSS3**: Styling and layout (including Flexbox and Grid).  
- **JavaScript**: Interactivity and DOM manipulation.
